## Ruby Language Center

This is an overview of the available tools and suggested practices for using Ruby with MongoDB. Those wishing to skip to more detailed discussion should check out the [Ruby Driver Tutorial](http://api.mongodb.org/ruby/current/file.TUTORIAL.html), [Getting started with Rails](http://www.mongodb.org/display/DOCS/Rails+-+Getting+Started) or [Rails 3](http://www.mongodb.org/display/DOCS/Rails+3+-+Getting+Started), and [MongoDB Data Modeling and Rails](http://www.mongodb.org/display/DOCS/MongoDB+Data+Modeling+and+Rails). There are also a number of good [external resources](http://www.mongodb.org/display/DOCS/Ruby+External+Resources) worth checking out.
