### 字符串对象

字符串对象用于处理已有的字符块。

**例子：**

下面的例子使用字符串对象的长度属性来计算字符串的长度。

<pre>
<javascript>
var txt="Hello world!"
document.write(txt.length)
</javascript>
</pre>

上面的代码输出为：

```javascript
12
```

下面的例子使用字符串对象的toUpperCase()方法将字符串转换为大写：

<pre>
<javascript>
var str="Hello world!"
document.write(str.toUpperCase())
</javascript>
</pre>

上面的代码输出为：

```javascript
HELLO WORLD!
```
